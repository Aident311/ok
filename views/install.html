Starter Botkit Discord

Remix this project

"Remixing" on Glitch makes a new copy of this project. It will have a randomly generated name which you can change.

Register your Discord application

The first step is to create a new Application on the Discord developer portal.

Get Your Token

Once you've created a Discord Application you should see an option on the sidebar labeled "Bot". Click on it. Now this is where you'll get your token. A token is a password, which means you want to keep it secure. If someone gets it they can act as your bot, which can case some issues. You don't want it in server.js because by default in Glitch anyone can see that file. And if you copy the project, it will copy the file with the secure token.

The only place you should put it is the file shown below .env

You should put your token after DISCORD_TOKEN=. So if your token is ewriewirew438234adadfs it will look like DISCORD_TOKEN=ewriewirew438234adadfs. Make sure you don't accidentally add any spaces or other characters after it because it won't work correctly.
